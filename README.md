This is a fork of [cadMousePro by GlobalCode](https://mouse.globalcode.info/v_0/cadMousePro.xhtml).

# Building

Setup the build environment. This only needs to be done once.

    make image

Build the daemon and gui applications.

    make build

# Running

Running the daemon using

    cadMousePro --daemon

periodically sends feature requests to the mouse so it can be used wired.
